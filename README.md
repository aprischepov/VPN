VPN

Проект запускается из xcode просто через cmd + r, дополнительных действий не требуется.
Минимальная версия: macOS 13.0+

Изменения UI, UX:
Переработан дизайн, добавлены акцентные цвета, приятнее глазу приложение, добавлена поддержка светлой и темной темы.
Выравнены отступы, приложение имеет определенный размер, без лишней информации, чтобы не занимать много места на рабочем столе любого макбука, в том числе и самых мелких диагоналей.
Дополнительные кнопки перенесены в навигационную панель, чтобы основную часть приложения вывести для главного контента.
Список серверов переработан, чтобы на главном экране отображалась только текущий сервер, а не все. Также добавлены флаги, чтобы глаз быстрее цеплялся за нужный регион.
